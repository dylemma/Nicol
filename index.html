Unleash the power of the Scala!

With Nicol, you can now use all the finest features of Scala to create your games.

Based on LWJGL, Nicol does not feel at all like a hijacked Java library, it makes full use of Scala's finest programming features, such like:

  • Pattern matching with case classes
  • Lists, Tuples and Functions used natively
  • Implicits whereever you don't wanna bother

So check out the Tutorial and dive into it today!
